# Tomato Focus

A small Pomodoro application where you can track your focus time while working on your special tasks.

It is developed with Phoenix Framework, LiveView and the power of Elixir language.

### Do it yorself

You can use this applciation. To start your Phoenix server:

  * Run `mix setup` to install and setup dependencies
  * Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
